ubuntu配置opencv

  • 网上所有的教程都无法直接成功，所以，个人的踩坑经历
  • 首先去opencv-python下载对应Python版本的opencv
  • 然后移入树莓派中，使用pip3 install ...whl即可成功
  • 很多的包都有这个特点，所以，要注意这个点，当自己的树莓派无法安装时，可以考虑这个专门为树莓派做的网站
